# 个人博客系统 python54

## 项目概述

个人博客系统是一个基于Python开发的网络应用，它使用Vue.js与HTML构建前端界面，后端则采用Flask框架，数据库使用MySQL进行数据存储。

## 技术栈

- 前端：HTML + Vue.js
- 后端：Python Flask
- 数据库：MySQL

## 功能模块

### 用户端

- 用户登录与注册
- 浏览与搜索博客文章
- 发布博客文章
- 对博客文章进行点赞与评论
- 关注博主
- 个人博客空间管理

### 管理员端

- 博客文章审核
- 增删改分类标签
- 查看博客发布情况
- 查看博客流量统计信息

## 系统角色

- 普通用户：可以浏览、发布、评论博客等。
- 管理员：负责审核博客内容，管理分类标签，查看统计分析数据。

## 运行环境

- 操作系统：兼容性良好，主流操作系统均支持
- Python版本：3.6及以上
- MySQL版本：5.7及以上

## 部署步骤

1. 安装Python环境
2. 安装MySQL数据库，并进行配置
3. 克隆或下载源码到本地
4. 安装依赖项 `pip install -r requirements.txt`
5. 运行数据库迁移 `flask db upgrade`
6. 启动应用 `flask run`

## 目录结构

```
.
├── app                 # 应用目录
│   ├── __init__.py     # 初始化文件
│   ├── models.py       # 数据模型
│   ├── views           # 视图目录
│   └── templates       # HTML模板
├── migrations          # 数据库迁移目录
├── venv                # 虚拟环境目录
├── requirements.txt    # 依赖列表
└── run.py              # 应用启动脚本
```
